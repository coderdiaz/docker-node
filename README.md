# docker-node

> The Node.js environment used by EGOIST

## Feature

- Basic on [docker-node](https://github.com/nodejs/docker-node) (latest)
- vim
- git
- Yarn

## Usage

```bash
# To run this image
docker run -it --rm 0x142857/node /bin/bash
```
